# HTTP Status Codes (Multi-language)

This project provides a comprehensive list of HTTP status codes in multiple languages.

## 📁 i18n structure

- `i18n/fa.json`: Persian translations
- `i18n/en.json`: English translations
- `i18n/ar.json`: Arabic translations
- `i18n/zh.json`: Chinese translations
- `i18n/es.json`: Spanish translations
- `i18n/fr.json`: French translations

## 🔍 CLI Usage with Language Support

```bash
# Search in Persian
python httpcode.py 404 --lang fa

# Search in English (default)
python httpcode.py 404 --lang en
```
