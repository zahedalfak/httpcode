# راهنمای کدهای وضعیت HTTP

کدهای وضعیت پاسخ HTTP نشان می‌دهند که آیا یک درخواست HTTP خاص با موفقیت تکمیل شده است یا خیر. پاسخ‌ها به پنج کلاس گروه‌بندی می‌شوند:

1.  **پاسخ‌های اطلاعاتی** (100–199)
2.  **پاسخ‌های موفقیت** (200–299)
3.  **تغییر مسیرها** (300–399)
4.  **خطاهای کلاینت** (400–499)
5.  **خطاهای سرور** (500–599)

## نحوه استفاده از این پروژه

شما می‌توانید از ابزار خط فرمان برای پیدا کردن سریع توضیحات هر کد استفاده کنید:

```bash
python httpcode.py 404 --lang fa
```

این دستور توضیحات کامل کد 404 را به زبان فارسی نمایش می‌دهد.
