# Руководство по кодам состояния HTTP

Коды состояния ответа HTTP указывают на то, был ли успешно выполнен конкретный HTTP-запрос. Ответы сгруппированы в пять классов:

1.  **Информационные ответы** (100–199)
2.  **Успешные ответы** (200–299)
3.  **Сообщения о перенаправлении** (300–399)
4.  **Ответы об ошибках клиента** (400–499)
5.  **Ответы об ошибках сервера** (500–599)

## Как использовать этот проект

Вы можете использовать инструмент CLI, чтобы быстро найти описания для любого кода:

```bash
python httpcode.py 404 --lang ru
```
