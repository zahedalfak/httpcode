# HTTP ステータスコード ガイド

HTTP レスポンス ステータス コードは、特定の HTTP リクエストが正常に完了したかどうかを示します。レスポンスは 5 つのクラスに分類されます。

1.  **情報レスポンス** (100–199)
2.  **成功レスポンス** (200–299)
3.  **リダイレクト メッセージ** (300–399)
4.  **クライアント エラー レスポンス** (400–499)
5.  **サーバー エラー レスポンス** (500–599)

## このプロジェクトの使い方

CLI ツールを使用して、任意のコードの説明をすばやく見つけることができます。

```bash
python httpcode.py 404 --lang ja
```

これにより、404 コードの詳細な説明が日本語で表示されます。
