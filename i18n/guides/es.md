# Guía de códigos de estado HTTP

Los códigos de estado de respuesta HTTP indican si una solicitud HTTP específica se ha completado con éxito. Las respuestas se agrupan en cinco clases:

1.  **Respuestas informativas** (100–199)
2.  **Respuestas exitosas** (200–299)
3.  **Mensajes de redirección** (300–399)
4.  **Respuestas de error del cliente** (400–499)
5.  **Respuestas de error del servidor** (500–599)

## Cómo usar este proyecto

Puedes usar la herramienta CLI para encontrar rápidamente descripciones de cualquier código:

```bash
python httpcode.py 404 --lang es
```

Esto mostrará la descripción completa del código 404 en español.
