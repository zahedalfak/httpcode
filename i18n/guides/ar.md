# دليل رموز حالة HTTP

تشير رموز حالة استجابة HTTP إلى ما إذا كان قد تم إكمال طلب HTTP بنجاح أم لا. يتم تجميع الاستجابات في خمس فئات:

1.  **استجابات إعلامية** (100–199)
2.  **استجابات ناجحة** (200–299)
3.  **رسائل إعادة التوجيه** (300–399)
4.  **استجابات أخطاء العميل** (400–499)
5.  **استجابات أخطاء الخادم** (500–599)

## كيفية استخدام هذا المشروع

يمكنك استخدام أداة CLI للعثور بسرعة على أوصاف لأي رمز:

```bash
python httpcode.py 404 --lang ar
```

سيؤدي هذا إلى عرض الوصف الكامل للرمز 404 باللغة العربية.
